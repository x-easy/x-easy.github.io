# x-easy Brand Assets Engine 🎨

這是 x-easy 品牌的**核心視覺資產庫**與**自動化生產引擎**。
此目錄獨立運作，不依賴主網站的其他部分，旨在提供一個「單一真值來源 (Single Source of Truth)」，確保所有輸出的品牌素材（Logo、圖示、標準色）皆符合 `1.25px` 極細美學標準。

## 📂 目錄結構

- **`index.html`**: 生產引擎控制台。
  - 部署後訪問網址：`https://x-easy.github.io/brand/`
  - 功能：視覺化檢視所有圖示、一鍵複製 SVG 代碼、一鍵生成標準化 Blog 落地頁模板。
  
- **`assets.json`**: 核心數據庫。
  - 定義了品牌色票、線條粗細 (`stroke-width: 1.25`) 以及所有 SVG 路徑數據。
  - 若要新增或修改圖示，**僅需編輯此檔案**，前端控制台會自動更新。

## 🚀 如何使用

1. **訪問控制台**：
   直接打開 GitHub Pages 對應網址 (e.g., `/brand/`)。

2. **獲取圖示**：
   點擊畫面上的任一卡片，SVG 代碼即會自動複製到剪貼簿，可直接貼入您的 HTML 專案中。

3. **生成專案**：
   點擊右上角的 **「下載部落格落地頁模板」**，即可獲得一份包含品牌 header 與標準樣式的 `.html` 檔案，開始撰寫新內容。

## 🛠 維護指南

若需新增圖示，請開啟 `assets.json` 並依照以下格式新增項目：

```json
"category_name": {
  "icon_name": "<svg viewBox='0 0 24 24' ...></svg>"
}
